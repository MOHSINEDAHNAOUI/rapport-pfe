\chapter{Présentation du projet}
%=====================================================================

\section{Contexte}

Le secteur hôtelier connaît une transformation digitale importante. Les clients recherchent
des solutions de réservation en ligne rapides, intuitives et personnalisées. Ce projet vise à
développer une application web intelligente permettant de gérer les réservations hôtelières
de manière optimisée.

\section{Objectifs du projet}

\begin{itemize}
  \item Faciliter le processus de réservation pour les clients ;
  \item Optimiser la gestion des chambres et des réservations pour l'hôtel ;
  \item Offrir une interface moderne et responsive ;
  \item Automatiser les processus administratifs.
\end{itemize}

\section{Périmètre}

\begin{itemize}
  \item Application web accessible via navigateur ;
  \item Interface client pour la recherche et réservation ;
  \item Interface administrateur pour la gestion de l'hôtel ;
  \item Système de paiement en ligne sécurisé ;
  \item Tableau de bord analytique.
\end{itemize}

\chapter{Spécifications fonctionnelles}
%=====================================================================

\section{Gestion des utilisateurs}

\subsection{Types d'utilisateurs}

\begin{itemize}
  \item \textbf{Client :} peut rechercher, réserver et gérer ses réservations ;
  \item \textbf{Administrateur :} gestion complète du système.
\end{itemize}

\subsection{Fonctionnalités d'authentification}

\begin{itemize}
  \item Inscription avec validation par e-mail ;
  \item Connexion sécurisée ;
  \item Récupération de mot de passe ;
  \item Gestion de profil utilisateur ;
  \item Historique des réservations.
\end{itemize}

% -------------------------------------------------------------------
\section{Gestion des chambres}

\subsection{Catégorisation}

\begin{itemize}
  \item Types de chambres : Simple, Double (lit à deux places), Twin (deux lits séparés), Familiale ;
  \item Équipements : WiFi, TV, climatisation, etc. ;
  \item Capacité d'accueil (appel téléphonique à la réception).
\end{itemize}

\subsection{Disponibilité}

\begin{itemize}
  \item Calendrier de disponibilité en temps réel ;
  \item Gestion des périodes de maintenance ;
  \item Système de blocage de chambres ;
  \item Gestion des réservations groupées.
\end{itemize}

% -------------------------------------------------------------------
\section{Système de réservation}

\subsection{Recherche intelligente}

\begin{itemize}
  \item Recherche par dates (entrée / sortie) ;
  \item Filtres avancés (prix, type, équipements) ;
  \item Tri des résultats (prix, popularité, avis) ;
  \item Affichage de la disponibilité en temps réel ;
  \item Système de recommandations basé sur l'historique.
\end{itemize}

\subsection{Processus de réservation}

\begin{itemize}
  \item Sélection du type de chambre ;
  \item Choix des services additionnels (petit-déjeuner, parking, spa, etc.) ;
  \item Récapitulatif de la réservation ;
  \item Paiement en ligne sécurisé (carte virtuelle) ;
  \item Confirmation par e-mail avec QR code ;
  \item Possibilité de modification / annulation (en cas d'annulation, le client paie \textbf{10\,\%} du prix total).
\end{itemize}

\subsection{Tarification intelligente}

\begin{itemize}
  \item Tarifs variables selon la saison ;
  \item Promotions selon la convention d'entreprise (OCP, Police, etc.) ;
  \item Tarifs spéciaux (week-end, longue durée avec pourcentage de réduction).
\end{itemize}

% -------------------------------------------------------------------
\section{Gestion des services}

\subsection{Services de base}

\begin{itemize}
  \item Petit-déjeuner (buffet, continental, en chambre) ;
  \item Service en chambre ;
  \item Parking ;
  \item Transfert aéroport.
\end{itemize}

\subsection{Services premium}

\begin{itemize}
  \item Spa et wellness ;
  \item Restaurant et bar ;
  \item Salle de sport ;
  \item Blanchisserie ;
  \item Organisation d'événements.
\end{itemize}

% -------------------------------------------------------------------
\section{Système de paiement}

\begin{itemize}
  \item Carte bancaire (Visa, Mastercard, etc.) ;

  \item Paiement à l'arrivée (avec garantie).
\end{itemize}

\section{Base de données}

\subsection{Tables principales}

\begin{itemize}
  \item \texttt{users} --- utilisateurs ;
  \item \texttt{rooms} --- chambres ;
  \item \texttt{room\_types} --- types de chambres ;
  \item \texttt{amenities} --- équipements ;
  \item \texttt{reservations} --- réservations ;
  \item \texttt{payments} --- paiements ;
  \item \texttt{services} --- services additionnels ;
  \item \texttt{reviews} --- avis ;
  \item \texttt{promotions} --- promotions ;
  \item \texttt{settings} --- paramètres système.
\end{itemize}

\subsection{Contraintes}

\begin{itemize}
  \item Relations bien définies (clés étrangères) ;
  \item Index sur les colonnes fréquemment interrogées ;
  \item Soft deletes pour l'historique ;
  \item Timestamps sur toutes les tables.
\end{itemize}

\chapter{Contraintes}

\section{Contraintes techniques}

\begin{itemize}
  \item Compatibilité navigateurs : Chrome, Firefox, Safari, Edge (dernières versions) ;
  \item Temps de chargement : $< 3$ secondes ;
  \item Disponibilité : 99,9\,\% (uptime) ;
  \item Support mobile et tablette obligatoire.
\end{itemize}

\section{Contraintes de performance}

\begin{itemize}
  \item Support de 1\,000 utilisateurs simultanés minimum ;
  \item Temps de réponse API $< 500$\,ms ;
  \item Base de données optimisée pour $100\,000$+ réservations.
\end{itemize}
